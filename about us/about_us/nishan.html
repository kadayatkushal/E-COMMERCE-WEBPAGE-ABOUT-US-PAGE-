my knowledge

HTML

HTML is the most commonly used markup language for Web pages. HTML is an abbreviation for Hyper Text Markup Language. It describes the layout of a Web page.

CSS & Bootstrap

The language we use to style an HTML document is CSS. CSS outlines the presentation of HTML elements. The most popular free CSS framework in the world is Bootstrap.

Javascript

JavaScript is the world's most popular programming language. JavaScript is the programming language of the Web.

jQuery

jQuery is a lightweight, feature-rich JavaScript library. It simplifies HTML document traversal and manipulation, event handling, animation, and Ajax with an easy-to-use API that works across a wide range of browsers.

React

React is a JavaScript library created by Facebook. It is a User Interface (UI) library. It is a tool for building UI components

SQL

SQL is a standard language for storing, manipulating and retrieving data in databases.
